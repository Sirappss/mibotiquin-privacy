Política de Privacidad — Mi Botiquín

Última actualización: 28 de marzo de 2026

Mi Botiquín es una aplicación gratuita desarrollada por SirAppsDev. Esta política de privacidad describe cómo se gestiona la información en la aplicación.

Datos que se almacenan

Todos los datos que introduces en la app (medicamentos, fechas de caducidad, tratamientos) se almacenan únicamente en tu dispositivo. No se envían a ningún servidor ni se comparten con terceros.

Permisos utilizados

  • Cámara: para escanear los códigos de barras y DataMatrix de los envases.
  • Ubicación: para localizar farmacias cercanas a tu posición. No se almacena ni se comparte.
  • Notificaciones: para avisarte de caducidades próximas y recordatorios de tratamiento.
  • Almacenamiento: para exportar e importar copias de seguridad del botiquín.

Servicios externos

La app consulta la base de datos pública de medicamentos de la Agencia Española de Medicamentos y Productos Sanitarios (AEMPS) a través de su API CIMA. Esta consulta no incluye ningún dato personal.

Para mostrar farmacias cercanas se utiliza OpenStreetMap y la API Overpass, servicios de mapas de código abierto.

Anuncios

La aplicación puede mostrar anuncios a través de Google AdMob. Google puede usar información del dispositivo para mostrar anuncios relevantes. Puedes consultar la política de privacidad de Google en policies.google.com/privacy.
